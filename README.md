# ukeban-front

## src 中的源代码
### App.js
渲染首页， 包括标签栏。标签栏用两种方式实现，一种是声明式导航（用了state来确定点击的标签，并转到相对应的路由），一种是编程式导航（用了NavLink）
### App.css
排版首页，但css我不怎么会写，可能逻辑很乱lol
### RouterIndex.js
路由模块，存储路由
### views
包含标签栏导航到的各个页面

## 遇到的问题
1. 写了两种标签栏实现方式，不知道哪种比较好。标签栏中字体的样式个原网站不一样。  
2. css不怎么会，基本上是用什么查什么，排版跟原网站差距较大
3. 代码基于react脚手架，不会用嵩哥给的脚手架
